========================================================
   SISTEMA DE ENTREGA POR DRONES - DTI DIGITAL
========================================================

📋 INSTRUÇÕES RÁPIDAS DE INSTALAÇÃO E EXECUÇÃO

1. INSTALAR DEPENDÊNCIAS:
   npm install

2. EXECUTAR O PROJETO:
   npm start

3. ACESSAR O SISTEMA:
   Abra o navegador em: http://localhost:8000

========================================================

📦 ESTRUTURA DO PROJETO:

   drone-delivery-system/
   ├── index.html         (Interface)
   ├── styles.css         (Estilos)
   ├── drone.js           (Classe Drone)
   ├── optimizer.js       (Algoritmos)
   ├── app.js             (Lógica principal)
   ├── tests.js           (Testes unitários)
   ├── server.js          (Servidor Node.js)
   └── package.json       (Configuração)

========================================================

🎮 COMO USAR:

   1. Configure o número de drones (1-10)
   2. Adicione pedidos ou gere aleatórios
   3. Clique em "▶ Iniciar" para começar
   4. Observe os drones coletando e entregando

========================================================

🧪 EXECUTAR TESTES:

   1. Abra o sistema no navegador
   2. Abra o Console (F12)
   3. Digite: runTests()

========================================================

   1. Execute: git init
